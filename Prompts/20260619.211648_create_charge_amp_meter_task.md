# User Prompt
Create a complete Android application in Java (NOT Kotlin) called "Charge Amp Meter".

## Purpose

The application should detect when a charger is connected and continuously measure and display the battery charging current in amperes (A) and milliamperes (mA) using Android's BatteryManager APIs.

## Main Requirements

### Main Screen

Display the following information:

1. Current Charging Value

   * Real-time charging current.
   * Show in mA and A.
   * Update every second.

2. Average Charging Value

   * Calculate the average charging current from the beginning of the measurement session.
   * Show in mA and A.

3. Measurement Duration

   * Display elapsed measurement time in:

     * Hours
     * Minutes
     * Seconds

4. Charger Status

   * Connected
   * Disconnected
   * Charging
   * Not Charging

5. Reset Button

   * A button labeled "Reset Measurement".
   * When pressed:

     * Reset average calculations.
     * Reset timer.
     * Clear all collected samples.
     * Start a fresh measurement session immediately.

### Charging Current Measurement

Use:

* BatteryManager.BATTERY_PROPERTY_CURRENT_NOW
* BatteryManager.BATTERY_PROPERTY_CURRENT_AVERAGE (when available)

Implement device compatibility handling because some devices return:

* Positive values
* Negative values
* Microamps instead of milliamps

Normalize values and display correctly.

### Data Sampling

* Sample current every 1 second.
* Store samples in memory.
* Average value should be calculated from all samples collected since the last reset.

Formula:

Average Current =
(Sum of all sampled currents) / (Number of samples)
