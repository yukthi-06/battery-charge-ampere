# Walkthrough - Freeze Views and Timer on Disconnect

I have updated the screen updates and timer controls on disconnect.

## Changes Made
### Activities
Modified [MainActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/05_Early_Stage/Battery_Charge_Ampere_APK_GITHUB/app/src/main/java/com/vypeensoft/chargeampmeter/MainActivity.java) inside:

## Verification Results
- Disconnect: `Duration` stops ticking, `Average Current` holds its last average, text color of both turns gray, while `Current Current` continues to update.
- Connect: `Duration` ticks again, new samples are factored into `Average Current`, text returns to standard white/black text layout.
