# Implementation Plan - Color Code Status Field

## Problem
The charger status text remained in standard text styling, and the disconnected indicator was in a neutral gray. The status text and icon should change color depending on connection state: green when connected/charging, and red when disconnected.

## Proposed Changes
1. Modify `colors.xml` to set `colorDisconnected` to a shade of red (`#FF5252`).
2. Modify `MainActivity.java` inside `updateChargerStatusUI()` to apply the status color (green when charging/connected, red when disconnected) directly to the status text view (`tvChargerStatus`) and tint the status icon (`ivStatusIcon`).
