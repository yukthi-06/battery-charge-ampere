# Implementation Plan - Side-by-Side Average Current and Battery Level

## Problem
The dashboard needs to show the battery level percentage side-by-side with the average charging current rather than displaying the average card in a single full-width row.

## Proposed Changes
1. Update `strings.xml` to define the label `card_title_battery_level`.
2. Modify `content_main.xml` to wrap the Average card in a horizontal `LinearLayout` of weight 2, and add a companion CardView for the Battery Level on the right side.
3. Modify `ChargingMonitor.java` to implement a `getBatteryLevel()` query method retrieving live percentage data.
4. Modify `MainActivity.java` to declare `tvBatteryLevel`, bind it in `onCreate()`, and dynamically set the percentage value in `performSampleUpdate()`.
