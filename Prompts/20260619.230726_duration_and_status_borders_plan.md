# Implementation Plan - Duration and Status Borders

## Problem
The Duration and Charger Status cards on the main dashboard did not render border strokes, diverging from the design styles of the other cards (Current, Average, Battery Level).

## Proposed Changes
Modify `content_main.xml` to add `app:strokeWidth="1dp"` and `app:strokeColor="@color/colorPrimaryDark"` to both:
1. `Duration Card`
2. `Charger Status Card`
