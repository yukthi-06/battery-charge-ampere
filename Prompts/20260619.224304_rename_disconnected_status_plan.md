# Implementation Plan - Rename Disconnected Status

## Problem
The status text value "Disconnected" has sub-optimal word wrapping characteristics on some layouts. Renaming it to "Not Connected" provides a better structural text size footprint.

## Proposed Changes
1. Update status strings in `ChargingMonitor.java` to return `"Not Connected"` instead of `"Disconnected"`.
2. Update the resource label string `status_disconnected` in `strings.xml` to `"Not Connected"`.
3. Update the layout text placeholder in `content_main.xml` to `"Not Connected"`.
